# A0 技术尖刺运行方式

当前代码已经包含：

- 猫「瞭望员」灰盒角色
- 全屏任意起点触控摇杆与 WASD / 方向键移动
- 「余烬连弩」自动锁敌单射，精准只影响伤害区间，不影响射击方向
- 「星环刃」双刃环绕近战
- 持续追逐的拾荒兽与带预警蓄力的冲锋兽
- 敌人、子弹固定容量对象池
- 受击无敌帧、生命、命中反馈、击杀和调试 HUD
- Web 构建使用全屏自适应模板，不显示 Creator 默认标题和页脚

## 场景接线

`assets/scenes/Main.scene` 已经创建，并在 Canvas 上挂载了 `GameBootstrap`。在 Cocos Creator 3.8.8 中打开该场景后可以直接点击预览，不需要手工摆放节点或组件。

所有战斗对象都会由 `GameBootstrap` 运行时生成，不需要手工摆放敌人、子弹或 HUD。

## 核心测试

在仓库根目录运行：

```powershell
npm run test:cocos-core
```

核心战斗代码不依赖 Cocos API，可以独立进行确定性测试；Cocos 层只负责输入、画面、音频和平台接口。
