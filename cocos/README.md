# Cocos game client

This directory contains the new Cocos Creator 3.8.8 + TypeScript game client.

- Engine: Cocos Creator 3.8.8 (3.8 LTS)
- Template: official 2D empty project
- Existing root HTML/JavaScript game: retained as the mechanics and balance sandbox
- Generated directories such as `library`, `temp`, `local`, `build`, and `native` are not committed

Open the project with:

```powershell
& 'D:\cocos-creator\Creator\3.8.8\CocosCreator.exe' --project 'E:\codex\game\cocos'
```
