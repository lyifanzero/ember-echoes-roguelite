# 余火回响

一个原创的俯视角自动射击 Roguelite 游戏原型，玩法灵感来自生存割草类游戏。

## 在线游玩

[通过 GitHub Pages 打开游戏](https://lyifanzero.github.io/ember-echoes-roguelite/)

[打开可交互产品结构蓝图](https://lyifanzero.github.io/ember-echoes-roguelite/blueprint.html)

网页支持 PC 键盘与移动端触控。每次推送到 `main` 分支后，GitHub Actions 会自动测试、构建并更新公开网页。

产品蓝图采用章节式讨论：可以直接修改蓝图文本、记录每章笔记、标记确认状态、保存关键选项，并导出 Markdown 讨论稿。修改内容默认保存在当前浏览器，不会影响玩家页面。

## 直接游玩

双击 `index.html` 即可运行，不需要安装依赖。也可以在本目录启动任意静态文件服务器：

```powershell
python -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`。

## 操作

- `WASD` 或方向键：移动
- `Tab`：随时查看角色属性、实体武器、守护装备与技能等级
- `P`：暂停 / 继续
- `M`：开启 / 关闭程序化音乐与音效
- 多种武器会自动按各自方向和攻击逻辑开火
- 手机端：触摸战场任意位置生成动态摇杆，右下按钮切换声音、查看构筑或暂停
- 拾取蓝色火种积累经验，升级获得技能点
- 闯关模式在每关结束后统一查看构筑并分配技能点

## 游戏模式

- **闯关模式**：10 个递进关卡，第 5、10 关为首领节点；每关约 28～58 秒
- **无尽模式**：敌潮持续增强，每 45 秒开启一次技能整备
- **四档难度**：休闲、标准、残酷、噩梦，分别影响敌人生命、伤害、速度、数量与经验收益

## 肉鸽构筑系统

- **自主角色选择**：开局可从 8 个完整公开的角色流派中选择，每个角色都有明确优势、负面限制和独立战斗形象
- **独立武器选择**：确定角色后，再从该角色适配的实体武器中自主选择第一件装备，角色与武器不再绑定
- **分支路线**：每关结束从 3 条路线中选择一条，主动权衡精英、恢复、军械库、经验或额外技能点
- **战场异变**：每关随机获得孵化狂潮、黑曜硬化、血月躁动、不稳定核心或余烬丰饶规则
- **强化品质**：技能、实体武器和守护装备会出现普通、稀有、史诗品质，一次分别提升 1～3 级
- **流派共鸣**：特定武器和技能组合可以激活交叉火网、永动刃舞、自动工坊、裂隙风暴等额外效果
- **研究档案**：跨局记录已使用原型、发现武器与守护装备、激活共鸣、远征次数和通关次数

## 精英宝箱与成就

- 精英怪死亡后掉落可拾取宝箱，开启时暂停战斗并提供三选一奖励
- 普通敌人会按幸运属性概率掉落两选一补给箱；幸运越高，出现概率越高（最高 5.5%）
- 普通敌人也会掉落绿色医疗芯片，低血量时获得额外掉落倾向
- 奖励包括大量经验、武器直接提升 3 级、史诗守护装备、永久经验倍率、额外技能点、完全恢复与幸运增益
- 成就跨局保存，包含 8 种初始原型通关、无伤通关、单武器通关、低血通关、护甲/伤害数值挑战和高难度通关
- 首页可查看完成数量、最高关卡、宝箱开启数和全部成就条件

## 已实现内容

- 9 种实体武器：余烬连弩、双生侧炮、星环刃、脉冲环、裂隙轨道枪、铸渣霰弹器、熔炉喷射器、电弧线圈与铸火炮塔
- 铸火炮塔会固定在部署位置独立锁敌，不会跟随角色移动；武器槽上限扩展为 6
- 开局选择页、战斗 HUD 和完整构筑面板都会显示武器实体外观，而非只有名称或符号
- 自动追踪、侧向齐射、环绕切割、贯穿重炮、短程霰弹、持续熔火、即时连锁和固定部署等独立攻击模式
- 炽晶护盾、守望无人机、棘甲外骨骼和棱镜圣衣 4 种守护装备，提供拦截、炮台、反击与屏障效果
- 追踪、蛇形、重甲、远射、冲锋、召唤、精英和首领等敌人 AI
- 敌人远程弹幕、冲锋预警、召唤幼体和首领环形射击
- 10 种属性技能与 9 条武器升级线
- 经验拾取、磁吸、生命、护甲与恢复属性
- 我方弹丸采用亮色光矛，敌方弹丸统一为红色黑芯菱形，并在 HUD 提供图例
- 战斗 HUD 展示实体武器、守护装备等级、敌人种类与待分配技能点
- 局间/结算面板展示完整角色属性、武器、守护装备和技能等级
- 构筑面板展示流派标签、共鸣条件、本关路线、随机异变和局种
- 受击反馈、粒子、伤害数字、屏幕震动及响应式界面
- Web Audio 程序化战斗音效与随关卡/首领强度变化的低频工业配乐，无外部音频版权依赖
- 硬边铸铁面板、冲压标签和克制光效组成的“余火工业”UI，减少玻璃拟态与泛用霓虹感

## 验证

```powershell
npm test
npm run balance
```

- `balance.js` 集中管理武器曲线、敌人基础值、十关生命成长、精准与暴击模型。
- 远程武器默认锁定具体敌人并持续修正弹道；攻击随机性只作用于伤害区间和暴击。
- 完整的原型后制作、程序、策划与美术方向见 `DESIGN_DIRECTION.md`。
