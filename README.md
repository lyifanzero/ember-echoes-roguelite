# 余火回响

一个原创的俯视角自动射击 Roguelite 游戏原型，玩法灵感来自生存割草类游戏。

## 直接游玩

双击 `index.html` 即可运行，不需要安装依赖。也可以在本目录启动任意静态文件服务器：

```powershell
python -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`。

## 操作

- `WASD` 或方向键：移动
- `Tab`：随时查看角色属性、武器库与技能等级
- `P`：暂停 / 继续
- 多种武器会自动按各自方向和攻击逻辑开火
- 手机端：左下虚拟摇杆移动，右下按钮查看构筑或暂停
- 拾取蓝色火种积累经验，升级获得技能点
- 闯关模式在每关结束后统一查看构筑并分配技能点

## 游戏模式

- **闯关模式**：10 个递进关卡，第 5、10 关为首领节点；每关约 28～58 秒
- **无尽模式**：敌潮持续增强，每 45 秒开启一次技能整备
- **四档难度**：休闲、标准、残酷、噩梦，分别影响敌人生命、伤害、速度、数量与经验收益

## 肉鸽构筑系统

- **随机原型选秀**：每局从 6 个原型中随机出现 3 个，改变初始武器、属性和成长方向
- **分支路线**：每关结束从 3 条路线中选择一条，主动权衡精英、恢复、军械库、经验或额外技能点
- **战场异变**：每关随机获得孵化狂潮、黑曜硬化、血月躁动、不稳定核心或余烬丰饶规则
- **强化品质**：技能和武器会出现普通、稀有、史诗品质，一次分别提升 1～3 级
- **流派共鸣**：特定武器和技能组合可以激活交叉火网、永动刃舞、不灭堡垒、群星坠落等额外效果
- **研究档案**：跨局记录已使用原型、发现武器、激活共鸣、远征次数和通关次数

## 精英宝箱与成就

- 精英怪死亡后掉落可拾取宝箱，开启时暂停战斗并提供三选一奖励
- 奖励包括大量经验、武器直接提升 3 级、永久经验倍率、额外技能点、完全恢复与幸运增益
- 成就跨局保存，包含 6 种初始原型通关、无伤通关、单武器通关、低血通关、护甲/伤害数值挑战和高难度通关
- 首页可查看完成数量、最高关卡、宝箱开启数和全部成就条件

## 已实现内容

- 余烬连弩、双生侧炮、星环刃和脉冲环 4 种武器，可重复升级
- 自动追踪、侧向齐射、环绕切割和全方向弹幕
- 追踪、蛇形、重甲、远射、冲锋、召唤、精英和首领等敌人 AI
- 敌人远程弹幕、冲锋预警、召唤幼体和首领环形射击
- 9 种属性技能与 4 条武器升级线
- 经验拾取、磁吸、生命、护甲与恢复属性
- 战斗 HUD 展示武器等级、敌人种类与待分配技能点
- 局间/结算面板展示完整角色属性、武器和技能等级
- 构筑面板展示流派标签、共鸣条件、本关路线、随机异变和局种
- 受击反馈、粒子、伤害数字、屏幕震动及响应式界面
